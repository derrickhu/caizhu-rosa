# 关卡重设计说明

本轮重设计目标：后期关卡从“短时间高分硬压”改为“长局高目标 + 可规划初始棋盘”。每关通过固定/半固定布局提供明确策略入口，随机刷子只作为变化补充。

## 设计原则

- 高分需要操作空间。后期关卡允许 3-5 分钟或 45-60 步，让玩家有观察、铺局和积累分数的时间。
- 每关至少有一个第一眼可见的策略入口，同时保留更高收益的进阶路线。
- 石块必须配可规划炸弹路径，不能只作为随机堵路。
- 冰冻和锁链应放在“解开后能形成更高分”的位置。
- 特殊棋子随机概率保持低，关卡差异主要由初始布局和开局下一批棋子承载。

## 后续调参检查点

- 第 10、20、27、30 关重点检查炸弹是否真的能破解关键石块。
- 第 8、9、18、23、28 关重点检查冰冻/锁链第一击后是否能形成后续收益。
- 第 24、25、29、30 关重点观察 300 秒是否足够沉浸但不拖沓。
- 若后期三星过高，优先调整星级倍率，而不是继续压缩时间或步数。
