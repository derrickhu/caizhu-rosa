# 彩珠五连 美术风格定义

## 核心风格
- **大胆几何图案** — 对角线条纹、菱形格、放射线等几何元素
- **矢量/平面设计** — 类似 Figma/Illustrator 输出的干净图形，非手绘
- **高饱和度色彩** — 鲜明、大胆、不褪色
- **边缘暗角** — 四角自然渐暗，聚焦中心
- **NO 手绘/水彩/插画感** — 不要 soft、dreamy、painterly

## 色板

### 经典模式（深沉大气）
- 主调：深海军蓝 #1B2838 ~ #2C3E50
- 条纹：#1F3044 / #253D54（深浅交替）
- 暖光点缀：中心微弱琥珀光 #C9A340 at 8% opacity
- 暗角：#0A0F18

### 关卡模式（明快活泼）
- 主调：珊瑚粉橘 #E86F5A ~ 浅粉 #F2A89A（参考截图的红色条纹）
- 或可替代方案：薄荷绿 #45B5A6 ~ 青绿 #3BA99A
- 条纹：两种相近色交替
- 暗角：中心亮、四角自然暗 15-20%

### 首页（高级感）
- 主调：深紫蓝 #1A1A3E ~ #2D2B55
- 装饰：彩色玻璃珠抽象圆形（红/蓝/绿/黄/紫/橙/青）
- 暗角：#0D0D20
