五、后续接入注意
1. 正式 UI 组件尽量不要包含文字，文字由 PIXI.Text 绘制。
2. 道具图标与棋子皮肤需要透明 PNG；若 Gemini 无透明背景，拆图后用 rembg 去背景。
3. 背景图不需要去背景，直接 JPG/PNG。
4. 所有合图先放在 ../game_assets/caizhu-rosa/assets/raw/，确认后再进入 split/nobg/final 流程。
